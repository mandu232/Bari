# BARI

Godot 4.6으로 제작 중인 박물관 건설 + 던전 로그라이크 게임입니다.

던전을 탐험해 유물을 수집하고, 박물관에 전시해 영력을 생산하세요.
쌓인 영력으로 시설을 확장하고 더 깊은 던전에 도전하세요.

---

## 게임 진행 방법

```
던전 탐험 → 유물 수집
      ↓
박물관 귀환 → 전시대 설치 및 유물 배치
      ↓
전시대가 영력(靈力) 자동 생산
      ↓
영력으로 시설 건설 / 강화
      ↓
더 깊은 던전 도전
```

---

## 핵심 시스템

### 박물관
- **전시대**를 설치해 수집한 유물을 전시
- 유물이 배치된 전시대는 **영력(靈力)** 을 지속적으로 생산
- **발전소**를 건설해 전시대에 전력 공급
- 전력이 없는 전시대는 유물 배치 및 상호작용 불가

### 전력 시스템
- 발전소는 설정된 **반경 내** 건물에만 전력 공급
- 발전소마다 **용량 한계**(`power_output`) 존재 — 초과 시 추가 건물은 미공급
- 발전소에 가까이 가면 공급 범위가 원으로 시각화
- 건설 모드에서 발전소와 전시대를 잇는 **연결선** 표시

| 단계 | 이름 | 공급 전력 | 공급 범위 |
|------|------|----------|----------|
| Lv.1 | 영화로 | 20 | 120px |
| Lv.2 | 혼화로 | 80 | 180px |
| Lv.3 | 신화로 | 180 | 260px |

### 건설 / 철거
- `B` 키로 **건설 메뉴** 열기
- 건물 선택 후 좌클릭으로 배치
- 배치 모드 중 우클릭 → 취소 후 건설 메뉴로 복귀
- 건설 메뉴가 열린 상태에서 설치된 건물에 **우클릭 → 철거**
  - 건설 비용의 **50%를 영력으로 환급**
  - 전시대 철거 시 배치된 유물은 인벤토리로 자동 반환

### 던전
- 박물관 던전 입구 포탈로 진입
- 적을 처치하고 유물 수집
- 박물관으로 귀환해 전시

## 조작법

| 행동 | 입력 |
|------|------|
| 이동 | `WASD` / 방향키 |
| 상호작용 | `E` |
| 공격 | 마우스 좌클릭 |
| 대시 | `Shift` |
| 건설 메뉴 열기 | `B` |
| 건물 배치 | 좌클릭 (배치 모드) |
| 배치 취소 | 우클릭 / `ESC` |
| 건물 철거 | 우클릭 (건설 메뉴 열린 상태) |
| 건설 메뉴 닫기 | `B` / `ESC` |

## 프로젝트 구조

```
bari/
├── AutoLoad/
│   ├── assets/          # 스프라이트, 폰트, SpriteFrames
│   └── scenes/          # 재사용 씬 파일 (.tscn)
├── resources/
│   ├── artifacts/       # ArtifactData 리소스 (.tres)
│   └── buildables/      # BuildableItem 설계도 (.tres)
└── scripts/
    ├── GameManager.gd   # AutoLoad 싱글톤 — 전역 상태 관리
    ├── artifact/        # ArtifactData 클래스
    ├── museum/          # Museum, BuildManager, PowerPlant, UI 스크립트
    ├── player/          # 플레이어 컨트롤러, 카메라
    └── spirit/          # 스피릿
```
