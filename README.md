# BARI

Godot 4.6으로 제작 중인 박물관 건설 + 던전 로그라이크 게임입니다.

던전을 탐험해 유물을 수집하고, 박물관에 전시해 영력을 생산하세요.
쌓인 영력으로 시설을 확장하고 더 깊은 던전에 도전하세요.

---

## 게임 진행 방법

```
던전 탐험 → 유물 수집
      ↓
박물관 귀환 → 전시대 설치 및 유물 배치
      ↓
전시대가 영력(靈力) 자동 생산
      ↓
영력으로 시설 건설 / 강화
      ↓
더 깊은 던전 도전
```

---

## 핵심 시스템

### 박물관
- **전시대**를 설치해 수집한 유물을 전시
- 유물이 배치된 전시대는 **영력(靈力)** 을 지속적으로 생산
- **발전기**를 건설해 전시대에 전력 공급
- 전력이 없는 전시대는 유물 배치 및 상호작용 불가

### 전력 시스템
- 발전기는 설정된 **반경 내** 건물에만 전력 공급
- 발전기마다 **용량 한계**(`power_output`) 존재 — 초과 시 추가 건물은 미공급
- 발전기에 가까이 가면 공급 범위가 원으로 시각화
- `[F]` 키로 **전선 연결** (발전기 ↔ 전시대, 발전기 ↔ 전력 공급기)
- **전력 공급기**를 중간에 배치해 더 먼 곳까지 전력 연장 가능

| 건물 | 역할 | 공급 전력 | 공급 범위 | 건설 비용 |
|------|------|----------|----------|---------|
| 발전기 Lv.1 | 전력 생산 | +40 | 150px | 20영력 |
| 전력 공급기 | 전력 중계 | 최대 60 중계 | 자동 120px / 배선 200px | 80영력 |

### 에코(Echo) 시스템
- 유물을 전시대에 배치하면 해당 유물에 깃든 **에코**가 소환됨
- 에코는 **출력 · 안정도 · 활성도** 세 가지 욕구(needs)를 가짐
- 에코의 기분이 좋을수록 영력 생산 배율이 높아짐
- 지원 시설로 에코의 욕구를 회복해 생산 효율을 유지

| 시설 | 회복 욕구 | 회복량(초당) | 효과 범위 | 전력 소비 | 건설 비용 |
|------|---------|------------|---------|---------|---------|
| 충전소 | 출력 | +3.5 | 100px | 15 | 80영력 |
| 홀로그램 분수 | 안정도 | +5.0 | 120px | 15 | 80영력 |
| 기록재생기 | 활성도 | +11.0 | 110px | 15 | 80영력 |

### 건설 / 철거
- `B` 키로 **건설 메뉴** 열기
- 건물 선택 후 좌클릭으로 배치
- 배치 모드 중 우클릭 → 취소 후 건설 메뉴로 복귀
- 건설 메뉴가 열린 상태에서 설치된 건물에 **우클릭 → 철거**
  - 건설 비용의 **50%를 영력으로 환급**
  - 전시대 철거 시 배치된 유물은 인벤토리로 자동 반환

### 건설 가능 시설 일람

| 시설 | 설명 | 건설 비용 | 전력 소비 |
|------|------|---------|---------|
| 박물관 본관 | 박물관 중앙 관리 시설 | 0 | — |
| 전시대 | 유물 전시 및 영력 생산 | 50영력 | 20 |
| 발전기 Lv.1 | 적은 양의 전력 생산 | 20영력 | — (생산: +40) |
| 전력 공급기 | 전력 중계 (최대 60) | 80영력 | — (중계) |
| 충전소 | 에코 출력 회복 (+3.5/초, 100px) | 80영력 | 15 |
| 홀로그램 분수 | 에코 안정도 회복 (+5.0/초, 120px) | 80영력 | 15 |
| 기록재생기 | 에코 활성도 회복 (+11.0/초, 110px) | 80영력 | 15 |

### 던전
- 박물관 던전 입구 포탈로 진입
- 적을 처치하고 유물 수집
- 박물관으로 귀환해 전시

## 조작법

| 행동 | 입력 |
|------|------|
| 이동 | `WASD` / 방향키 |
| 상호작용 | `E` |
| 공격 | 마우스 좌클릭 |
| 막기 | 마우스 우클릭 |
| 대시 | `Shift` |
| 전선 연결 | `F` (발전기·전력 공급기 근처) |
| 건설 메뉴 열기 | `B` |
| 건물 배치 | 좌클릭 (배치 모드) |
| 배치 취소 | 우클릭 / `ESC` |
| 건물 철거 | 우클릭 (건설 메뉴 열린 상태) |
| 건설 메뉴 닫기 | `B` / `ESC` |

## 프로젝트 구조

```
bari/
├── AutoLoad/
│   ├── assets/          # 스프라이트, 폰트, SpriteFrames
│   └── scenes/          # 재사용 씬 파일 (.tscn)
├── resources/
│   ├── artifacts/       # ArtifactData 리소스 (.tres)
│   ├── buildables/      # BuildableItem 설계도 (.tres)
│   └── talismans/       # TalismanData 리소스 (.tres)
└── scripts/
    ├── GameManager.gd   # AutoLoad 싱글톤 — 전역 상태 관리
    ├── artifact/        # ArtifactData 클래스
    ├── museum/          # Museum, BuildManager, PowerPlant, UI 스크립트
    ├── player/          # 플레이어 컨트롤러, 카메라
    └── spirit/          # 에코(Echo) 시스템
```
